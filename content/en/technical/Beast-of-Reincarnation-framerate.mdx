<Section icon={Target} title="Step-by-Step Launcher and Compatibility Setup">

Configuring your game launcher properties ensures that the operating system grants the game administrative privileges and bypasses unnecessary full-screen optimization layers that cause micro-stutters.

<Steps>
  <Step num="1" title="Access Launcher Properties">
    Locate your game launcher (such as Steam), right-click the icon, and select **Properties**.
  </Step>
  <Step num="2" title="Modify Compatibility Settings">
    Navigate to the **Compatibility** tab. Check the box to **Run this program as an administrator**.
  </Step>
  <Step num="3" title="Disable Fullscreen Optimizations">
    Check the box to **Disable fullscreen optimizations**. This forces Windows to let the game manage its own display presentation, reducing input delay.
  </Step>
  <Step num="4" title="Apply and Restart">
    Click **Apply**, then **OK**. Restart your PC to ensure all administrative permissions register correctly before launching.
  </Step>
</Steps>

| Compatibility Setting | Status | Objective |
| :--- | :--- | :--- |
| **Run as Administrator** | Enabled | Grants full access to system resources |
| **Disable Fullscreen Optimizations** | Enabled | Reduces micro-stutters and input lag |
| **Compatibility Mode** | Windows 8 (Optional) | Resolves legacy engine crashes |

<Callout type="info" title="Admin Mode">
Running your launcher as an administrator ensures that background software cannot block the game from accessing essential hardware threads.
</Callout>

</Section>

<Section icon={Shield} title="Background Applications and Task Manager Tuning">

Running too many background tasks is the primary cause of sudden framerate drops. You must clean up your startup list and adjust process priorities to guarantee smooth gameplay.

| Process Action | Recommended Setting | Resource Saved |
| :--- | :--- | :--- |
| **Xbox Game Bar** | Disabled | Background CPU / RAM |
| **Background Recording** | Disabled | GPU Encoder overhead |
| **Startup Apps** | Disable all non-essential items | System boot memory |
| **Game Priority** | High or Above Normal | CPU Core scheduling |

<Callout type="success" title="Priority Tip">
Setting your game process priority to 'High' in the Task Manager details tab tells your CPU to process game instructions before handling background system tasks.
</Callout>

</Section>
