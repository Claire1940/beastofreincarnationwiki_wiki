<Section icon={Shield} title="対応端末とハードウェア動作環境">

快適なゲームプレイ体験を確保するため、お使いの端末が要求スペックを満たしているか事前に確認してください。高品質なテクスチャや複雑な物理演算を処理するには、最新のプロセッサと十分なメモリが必要となります。非対応の端末でゲームを実行すると、頻繁なクラッシュ、描画のバグ、または端末の発熱を引き起こす可能性があります。

モバイルプレイヤーの方は、OSが最新バージョンにアップデートされていることを確認してください。古いシステムソフトウェアは、ゲームのセキュリティプロトコルや最適化機能と競合する場合があります。以下は、AndroidおよびiOS端末の公式スペック情報です。

| プラットフォーム | 最低スペック | 推奨スペック |
|---|---|---|
| Android | Snapdragon 845, 4GB RAM, Android 9 | Snapdragon 8 Gen 1, 8GB RAM, Android 12 |
| iOS | iPhone 11, 4GB RAM, iOS 14 | iPhone 14 Pro, 6GB RAM, iOS 16 |

デスクトップ環境でのプレイを好む方向けに、PCクライアントではより高画質なグラフィックス、高いフレームレート、そしてカスタマイズ可能なキーバインドが提供されます。お使いのPCが動作要件を満たしているか、以下の表でご確認ください。

| ハードウェア | 最低動作環境 | 推奨動作環境 |
|---|---|---|
| CPU | Intel Core i5-6400 / AMD Ryzen 3 1200 | Intel Core i7-8700 / AMD Ryzen 5 3600 |
| GPU | NVIDIA GTX 1050 / AMD RX 560 | NVIDIA GTX 1660 Ti / AMD RX 5600 XT |
| RAM | 8 GB RAM | 16 GB RAM |
| ストレージ | 20 GB 以上の空き容量 (HDD) | 20 GB 以上の空き容量 (SSD) |
| OS | Windows 10 64-bit | Windows 11 64-bit |

<Callout type="warning" title="ストレージに関する警告">
ファイルの展開プロセスをスムーズに行うため、インストールサイズとは別に、ストレージに少なくとも5GB以上の追加の一時的な空き容量があることを確認してください。
</Callout>

<Section icon={Target} title="ビースト・オブ・リンカネーションの事前インストール手順">

クライアントのセットアップには、アカウント登録、インストーラーの取得、およびゲームファイルのパッチ適用が含まれます。以下の手順に従って、選択したプラットフォームでのインストールを正常に完了させてください。ファイルの破損を防ぐため、各ステップを慎重に進めてください。

PCプレイヤーの方は、手動ダウンロードよりも効率的にファイルの検証やパッチ適用を行える公式ランチャーの使用を強くお勧めします。モバイルユーザーの方は、安全で正規品のアプリを入手するために、必ず公式のアプリストアからダウンロードしてください。

<Callout type="tip" title="ダウンロード速度を上げるコツ">
もしダウンロード速度が遅い場合は、DNS設定を変更するか、ダウンロードを一度一時停止してから再開して接続をリフレッシュしてみてください。
</Callout>

<Steps>
  <Step num="1" title="オンラインで事前登録">
    クライアントが配信される前に、公式ゲームページまたはストアの製品ページにアクセスし、メールアドレスを登録してアカウント情報を確保しておきます。
  </Step>
  <Step num="2" title="インストーラーのダウンロード">
    App Store、Google Play ストア、またはPC用の公式ランチャーサイトから、公式セットアップクライアントをダウンロードします。
  </Step>
  <Step num="3" title="事前インストールの実行">
    アプリを起動し、正式サービス開始日よりも前に、コアアセット、ボイスファイル、およびパッチファイルのダウンロードを開始します。
  </Step>
  <Step num="4" title="アカウントの連携">
    データを保護し、異なるプラットフォーム間でもプレイできるように、ゲームプロファイルをサードパーティのログインサービスに連携します。
  </Step>
</Steps>

インストール中に問題が発生した場合は、以下のトラブルシューティングガイドを参照して解決してください。

| エラーメッセージ | 考えられる原因 | 推奨される解決策 |
|---|---|---|
| 検証に失敗しました (Verification Failed) | ダウンロードファイルの破損 | クライアントを再起動するか、ファイルの整合性を確認する |
| 容量が不足しています (Insufficient Space) | ハードディスクが満杯 | 一時ファイルを削除して空き容量を確保する |
| タイムアウトしました (Network Timeout) | 不安定なインターネット接続 | 有線接続に切り替えるか、ルーターを再起動する |
| アプリが対応していません (App Not Compatible) | 古いOSバージョン | OSをアップデートするか、ハードウェアの互換性を確認する |
