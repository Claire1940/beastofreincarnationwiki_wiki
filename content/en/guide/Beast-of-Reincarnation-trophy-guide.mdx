<Section icon={Swords} title="Combat and Skill Mastery">

To earn the combat-related trophies in this **Beast of Reincarnation trophy guide**, you must master the synergy between Emma’s real-time katana actions and Koo’s command menu. The game rewards technical play, specifically the **Just-Guard** mechanic which generates points for Koo’s abilities.

<Steps>
  <Step num="1" title="Master the Just-Guard">
    Perform 50 Just-Guards to unlock the 'Impenetrable' trophy. This requires pressing the block button exactly as an attack lands.
  </Step>
  <Step num="2" title="Command Synergy">
    Spend 500 Command Points on Koo’s techniques. Open the command menu (Triangle on PS5) to slow time and select a skill.
  </Step>
  <Step num="3" title="Assassination Expert">
    Use Emma's plant-like hair to perform 20 falling assassinations from elevated positions.
  </Step>
  <Step num="4" title="Skill Tree Completion">
    Fully upgrade both Emma and Koo’s skill trees. This requires farming Amber and defeating Nushi.
  </Step>
</Steps>

<Callout type="tip" title="Farming Tip">
The 'Amber Hoarder' trophy requires 1,000,000 Amber. You can speed this up by equipping Spirit Stones that increase currency drops from malefacts found in dense blighted forests.
</Callout>
