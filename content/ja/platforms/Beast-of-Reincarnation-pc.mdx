<Section icon={Sparkles} title="PC最適化 ＆ グラフィック設定">

安定した60 FPSを維持するには、ビジュアルの美しさとシステムのパフォーマンスのバランスを取る必要があります。いくつかのグラフィック設定はパフォーマンスに大きな影響を与えるため、これらを調整することで、ゲーム全体の美観を損なうことなく大幅にフレームレートを向上させることができます。

| 設定項目 | 推奨値 | パフォーマンスへの影響 |
| --- | --- | --- |
| 影の品質 (Shadow Quality) | 中 (Medium) | 高 |
| ボリュメトリックフォグ (Volumetric Fog) | 低 または 中 | 高 |
| レイトレーシング (Ray Tracing) | オフ (Off) | 極めて高 |
| テクスチャの品質 (Texture Quality) | 高 (High)（VRAM 8GB以上推奨） | 低 |
| アンチエイリアス (Anti-Aliasing) | TAA または DLSSクオリティ | 中 |

影（シャドウ）とボリュメトリック効果は、森林エリアやダンジョンでのフレームレート低下の主な原因となります。これらの設定を1段階下げるだけで、全体的なパフォーマンスが最大15%改善される場合があります。グラフィックスカードがハードウェア超解像（アップスケーリング）に対応している場合は、高解像度でのフレームレート向上のためにDLSS、FSR、またはXeSSを有効にすることを強く推奨します。

<CardGrid cols={3}>
  <Card title="最大パフォーマンス優先プロファイル">
    - **対象ハードウェア**: 旧世代のGPU（GTX 10シリーズ、RX 500シリーズなど）
    - **主な調整**: 影を「低」に設定、モーションブラーを無効化、FSRを「パフォーマンス」に設定。

  </Card>

    - **期待される効果**: 4K解像度で100+ FPS。
  </Card>
</CardGrid>

視野角（FOV）スライダーの調整もパフォーマンスに影響を与えます。FOVを広げると、エンジンは一度に多くのオブジェクトを画面上に描画する必要があるため、CPU性能が低いシステムではボトルネックが発生しやすくなります。CPU起因のカクつきが発生する場合は、FOVをデフォルト値の近くに維持してください。

<Callout type="tip" title="アップスケーリングのコツ">
DLSSやFSRを使用する場合、4K解像度でプレイしている場合を除き、「ウルトラパフォーマンス」モードは避けることをお勧めします。1080pなどの低解像度でこのモードを使用すると、画面が過度にぼやけたり、細部の輪郭にチラつき（シュリンクノイズ）が発生したりする原因になります。
</Callout>

<Section icon={Swords} title="操作方法 ＆ キーバインド設定">

本作の戦闘システムは、正確なタイミング、回避、そして複雑なスキルのローテーションを駆使して戦います。操作レイアウトをお好みに合わせてカスタマイズすることで、手強いボス戦での戦闘効率や反応速度を劇的に向上させることができます。

| アクション | キーボード ＆ マウス | コントローラー (Xbox / PlayStation) |
| --- | --- | --- |
| キャラクター移動 | WASD | 左スティック |
| 弱攻撃 | 左クリック | X / □ ボタン |
| 強攻撃 | 右クリック | Y / △ ボタン |
| 回避 / ダッシュ | スペースキー | B / 〇 ボタン |
| 輪廻スキル (Reincarnation) | Q | LB + RB / L1 + R1 |
| インタラクト | E | X / □ ボタン |

キーボード＆マウスの構成は、瞬時のカメラ操作が可能なため、開けたエリアで複数の敵を追跡する際に非常に有利です。一方で、コントローラーのアナログ入力は優れた移動精度を提供するため、狭い足場を移動したり、敵の範囲攻撃（AoE）を回避したりするのが容易になります。

キーボード＆マウスを使用するプレイヤーは、回避アクションをマウスのサイドボタンに割り当てることで、親指をジャンプなどの他の移動操作に集中させることができます。また、アクティブスキルをWASDに近いキー（R、F、Cなど）にバインドすると、移動キーから指を離さずにスキルを発動できるようになります。

<Callout type="info" title="コントローラーのサポート">
PC版は、標準的な最新のコントローラーに対してネイティブなプラグアンドプレイに対応しています。PlayStationコントローラーを使用している場合、ボタンガイドが正しく表示されないときは、Steamのゲームプロパティから「Steam入力を有効にする」を設定してください。
</Callout>

起動時にクラッシュが発生する場合は、お使いのPCが最小動作環境を満たしているか確認してください。リリース当日のクラッシュの多くは、OSのアップデートが未適用であることや、DirectXコンポーネントの不足が原因です。ゲームクライアントを「管理者として実行」することで、セキュリティソフトとの権限競合が解決する場合もあります。

また、画面のちらつきやズレ（スクリーンテアリング）が発生する場合は、ゲームのフレームレートとモニターのリフレッシュレートが同期していない可能性があります。設定メニューで「垂直同期（V-Sync）」を有効にするか、GPUのコントロールパネルでフレームレート上限を設定することで、描画が滑らかになります。

<Callout type="warning" title="オーバーレイの競合">
Discord、GeForce Experience、RivaTunerなどのアプリケーションによるオーバーレイ表示は、ゲームの安定性低下、カクつき、または入力遅延の原因となる場合があります。予期しないパフォーマンス低下が発生した場合は、これらのオーバーレイを1つずつ無効にして原因を特定してください。
</Callout>
