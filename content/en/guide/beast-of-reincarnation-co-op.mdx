The combat system is designed to be highly tactical. Success depends on how effectively you utilize Koo during engagements. By parrying enemy attacks, you build up a resource gauge, which allows you to slow down time and select specialized blooming arts to turn the tide of battle.

| Feature | Description |
| --- | --- |
| Combat Style | Real-time action mixed with turn-based tactical elements |
| Companion Role | Koo acts as a programmable support/attacker |
| Resource System | Parrying builds the gauge for blooming arts |
| Core Mechanic | Time-slowing tactical menu for special moves |

<Callout type="tip" title="Strategy Note">
Always prioritize parrying when facing larger threats. Building Koo's gauge early allows you to unleash high-damage blooming arts that can stagger even the most formidable enemies.
</Callout>
