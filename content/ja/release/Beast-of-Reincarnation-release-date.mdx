<Section icon={Target} title="コアゲームプレイ：ピュリファイアー vs ヌシ">

Beast of Reincarnation の物語の核となるのは、ヌシに対するピュリファイアーの永遠の闘争です。告知トレーラーで明かされたように、任務はシンプルです。ヌシを見つけ出し、倒すこと。しかし、これらの生物は静止した存在ではないため、ゲームプレイは一筋縄ではいきません。彼らは環境や受けたダメージに基づいて進化し、すべての遭遇をユニークなものにします。

戦闘システムは、テンポの速いハックアンドスラッシュのメカニクスと戦術的なポジショニングを組み合わせています。プレイヤーは「輪廻ゲージ（リンカネーション・ゲージ）」を管理する必要があり、これを使用することで復活したり、強力なフィニッシュムーブを繰り出したりできますが、代償として世界の状況が永続的に変化します。

| メカニクス | 説明 | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| ヌシの進化 | 敵が戦闘中に変身する | 動的な戦術変更が必要 |
| ソウル・サイフォニング | 倒した敵のコアを吸収する | 新しいスキルツリーの分岐を解放 |
| ピュリファイアー・アーツ | クラス固有の特殊能力 | 高ダメージだがクールダウンが長い |
| 環境シナジー | 地形を利用して獣を罠にかける | 高ランクのボス戦で不可欠 |

<Callout type="success" title="戦闘の洞察">
ヌシの進化はランダムではありません。火属性の攻撃を頻繁に使用すると、ヌシは耐熱外殻を進化させる可能性があり、物理ダメージや氷属性ダメージへの切り替えを余儀なくされます。
</Callout>

<Section icon={Shield} title="システム要件と最適化">

Beast of Reincarnation を最高の状態で楽しむためには、特に2026年の発売であることを考慮すると、最新のハードウェアが必要になります。このゲームは、大規模な破壊可能環境と高精細なクリーチャーモデルを処理するために設計された独自のエンジンを使用しています。

開発者はPCプレイヤー向けに予備的な要件リストを提供しています。PS5およびXbox Series Xのコンソールプレイヤーは、「パフォーマンスモード」（60 FPSターゲット）と「解像度モード」（ネイティブ4Kターゲット）から選択できるようになります。

| コンポーネント | 最小要件 (1080p/30) | 推奨要件 (1440p/60) |
| :--- | :--- | :--- |
| OS | Windows 11 | Windows 11 |
| CPU | Intel i5-12400 / Ryzen 5 5600 | Intel i7-14700 / Ryzen 7 7800X3D |
| GPU | RTX 3060 / RX 6700 | RTX 4070 Ti / RX 7900 XT |
| RAM | 16 GB | 32 GB |
| ストレージ | 100 GB SSD | 100 GB NVMe SSD |

<Callout type="info" title="技術的な注意点">
DirectX 12 Ultimate が必須要件です。ハードウェア加速レイトレーシングを搭載していない古いGPUでは、「暗黒領域（Darkened Realms）」ゾーンで大幅なパフォーマンス低下が発生する可能性があります。
</Callout>

<Steps>
  <Step num="1" title="初期インストール">
    ゲームクライアント（約100 GB）をダウンロードします。最適なロード時間のために、SSDに十分な空き容量があることを確認してください。
  </Step>
  <Step num="2" title="キャラクター作成">
    ピュリファイアーの外見をカスタマイズし、初期の武器習熟度を決定する「ソウルパス」を選択します。
  </Step>
  <Step num="3" title="チュートリアルゾーン">
    プロローグ「目覚め」を完了し、魂の吸収（サイフォニング）とヌシの足跡追跡の基本を学びます。
  </Step>
  <Step num="4" title="ワールド探索">
    オープンワールドに足を踏み入れ、最初の狩りを開始します。進行状況を保存するために「輪廻の祠（Shrines of Reincarnation）」で休息することを忘れないでください。
  </Step>
</Steps>

</Section>
