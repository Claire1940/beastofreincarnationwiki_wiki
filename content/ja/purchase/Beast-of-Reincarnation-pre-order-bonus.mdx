<Section icon={Swords} title="スタンダードエディション：必須の予約特典">

ミニマリストなアプローチを好むプレイヤーにとって、スタンダードエディションは『Beast of Reincarnation』の核となる体験をすべて提供します。しかし、8月4日の発売前に購入を確定させることで、後から購入したプレイヤーには手に入らない特定のアイテムをアンロックできます。スタンダード版の主な**予約特典**は、主人公エマをサポートする相棒キャラクター、クゥに焦点を当てたものです。

<CardGrid cols={2}>
  <Card title="茶柴スキン (クゥ)">

  </Card>
  <Card title="30,000 アンバー">
    ゲーム内の主要な通貨です。30,000という量は控えめですが、序盤の装備アップグレードや基本的な回復アイテムの購入に役立ちます。
  </Card>
</CardGrid>

これらの報酬は一部のテストプレイヤーからは「ゲームバランスを壊すほどではない」と評されていますが、見た目の華やかさを提供してくれます。特にアンバーは、ボス戦や自然な進行でより大きな金額を稼げるようになるまでの、非常に初期の段階におけるちょっとしたブーストとして機能します。

| 特典アイテム | カテゴリ | 影響 | 入手可能性 |
| :--- | :--- | :--- | :--- |
| **茶柴スキン** | コスメティック | 見た目のみ | 予約限定 |
| **30,000 アンバー** | 通貨 | わずかな進行補助 | 予約限定 |
| **ゲーム本編** | ソフトウェア | フル体験 | 永続 |

<Callout type="tip" title="経済的なヒント">
30,000アンバーをすぐに消耗品に使ってしまわないようにしましょう。最初に遭遇する鍛冶屋まで取っておき、エマの初期武器をアップグレードすることで、より大きな戦闘上の利点を得ることができます。

<Section icon={Sparkles} title="デラックスエディション：限定装備とコスメティック">

デラックスエディションは、『Beast of Reincarnation』の最も完全なバージョンを求めるコアなファン向けに設計されています。スタンダードエディションよりも約10ドル（約1,500円）ほど高く設定されており、旅の視覚的・機能的な側面を強化する複数の限定アイテムが同梱されています。

<Steps>
  <Step num="1" title="北斗の剣を装備する">
    この武器はデラックス版限定です。帽子とは異なり、この剣は初期装備を上回る基本ステータスを持っていることが期待されており、序盤の数章でプレイヤーにわずかな優位性を与えます。
  </Step>
  <Step num="2" title="鬼の帽子を被る">
    エマ専用のコスメティックアイテムです。防御ステータスは提供しないようですが、「輪廻」や伝承というテーマに完璧に合致し、カットシーンで独特のシルエットを演出します。
  </Step>
  <Step num="3" title="黒柴スキンをアンロックする">
    茶柴スキンに加え、デラックス版の所有者はクゥ専用の「黒柴スキン」も受け取れます。これにより、探索中の相棒の見た目にさらなるバリエーションが加わります。
  </Step>
  <Step num="4" title="苗を管理する">
    「野菜の苗」はユニークな追加要素です。正確な機能はまだ調査中ですが、ゲーム内の拠点（ハブワールド）における拠点構築や料理メカニズムに関連している可能性が高いです。
  </Step>
</Steps>

デラックスエディションの最大の魅力は、通貨の所持量が大幅に増えることです。**合計100,000アンバー**（予約特典の30,000を含む）を受け取ることで、最初の主要なボス戦を前に強力な経済的基盤を築くことができます。

| デラックス特典アイテム | カテゴリ | メリット |

| **鬼の帽子** | コスメティック | エマ専用のユニークな頭装備 |
| **北斗の剣** | 武器 | 序盤のステータスブーストの可能性 |
| **100,000 アンバー** | 通貨 | 強力な購買力 |
| **野菜の苗** | 消耗品 | 農業・リソース生成 |

<Callout type="info" title="エキスパートの視点">
「北斗の剣」はおそらく「繋ぎ」の武器になるでしょう。初期装備よりは強力ですが、中盤でクラフトできる武器には追い抜かれるはずです。その価値は、プレイ開始から5〜10時間をよりスムーズに進められる点にあります。
</Callout>

<Section icon={Shield} title="対応プラットフォームとGame Passへの対応">

『Beast of Reincarnation』は幅広いプラットフォームで展開され、ほぼすべてのゲーマーがアクセスできるようになっています。2026年の発売における大きなハイライトは、発売初日から**Xbox Game Pass**および**PC Game Pass**に対応することです。

**サポートされているプラットフォーム:**
- Xbox Series X および Series S
- PlayStation 5 (PS5)
- Steam (PC)
- Xbox Cloud Gaming
- Windows PC (Microsoft Store経由)

Game Pass加入者の場合、ゲーム本編はサブスクリプションに含まれます。ただし、**予約特典**やデラックスアイテムは、サービス経由でプレイする場合、通常は別途「デラックス・アップグレード」の購入が必要になる点に注意してください。Microsoftは、これらの特典を含む「Game Passバンドル」を割引価格で提供するかどうかについては、まだ発表していません。

| プラットフォーム | Game Pass初日対応 | パッケージ版 | デジタル予約 |
| :--- | :--- | :--- | :--- |
| **Xbox Series X** | はい | あり | あり |
| **PlayStation 5** | いいえ | あり | あり |
| **Steam** | いいえ | なし | あり |
| **PC (Windows)** | はい | なし | あり |

<Callout type="success" title="プラットフォーム間の公平性">
コレクターにとって朗報なのは、現時点で**プラットフォーム限定アイテムは存在しない**ということです。PS5でプレイしてもXboxでプレイしても、予約さえすれば同じ柴犬スキンとアンバーのボーナスを受け取ることができます。
</Callout>

</Section>

最初の村に到着しても報酬が表示されない場合は、プラットフォームの「ゲームコンテンツの管理」セクションを再確認してください。時折、**予約特典**ファイルが個別の小さなダウンロードとしてリストされており、手動でインストールを開始する必要がある場合があります。

| 報酬タイプ | 受け取り場所 | タイミング |
| :--- | :--- | :--- |
| **通貨 (アンバー)** | ゲーム内メニュー | プロローグ終了後 |
| **スキン (柴犬)** | カスタマイズタブ | 即時 |
| **装備 (剣・帽子)** | 装備メニュー | プロローグ終了後 |
| **苗** | インベントリ・倉庫 | 最初の拠点到着後 |

<Callout type="info" title="パッケージ版に関する注意">
Amazonや店舗などでパッケージ版を購入した場合、予約特典コードはケースの中ではなく、レシートや納品書に印刷されている、あるいはメールで届く場合があります。捨てる前に必ず確認してください。
