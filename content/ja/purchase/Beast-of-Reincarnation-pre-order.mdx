<Section icon={Map} title="ゲームプレイ、設定、世界観の概要">

物語の舞台は西暦4026年、荒廃した2,000年後の未来です。世界は「ブライト（汚染）」と呼ばれる謎の力によって引き裂かれました。プレイヤーは、ブライトボーン（汚染されし者）として呪われ、追放されたエマとなり、不気味で美しいポストアポカリプス後の日本の廃墟を探索します。エマはこの旅で一人ではありません。探索や戦闘をサポートしてくれる忠実な犬の相棒、クゥが同行します。

エマとクゥは力を合わせ、危険な敵と戦い、古代の秘密を解き明かし、伝説の「ビースト・オブ・リンカネーション（転生の獣）」を探し出さなければなりません。この神話的な存在を見つけることが、残されたすべての生命が飲み込まれる前にブライトを終わらせるための、世界の最後の希望です。エマとクゥの絆は、ゲームプレイ体験の感情的かつメカニカルな核心を形成しています。

| ゲーム要素 | メカニクスの詳細 |
| :--- | :--- |
| **戦闘システム** | 近接武器とブライト能力を駆使したアクションRPG |
| **コンパニオンシステム** | クゥによる追跡、戦闘中の注意逸らし、パズル解決のサポート |
| **世界探索** | 植物に覆われた日本の廃墟や毒性ゾーンを巡るオープンワールド探索 |
| **拠点構築** | 苗を育て、リソースを管理してセーフゾーンをアップグレード |

<Callout type="info" title="コンパニオンのカスタマイズ">
柴犬スキンのようなコスメティックアイテムは、クゥの戦闘ステータスや能力を変化させるものではありません。これらは純粋に相棒を自分好みにするためのビジュアルアップグレードです。
</Callout>

<Section icon={Shield} title="PC動作環境とパフォーマンス予測">

本作はUnreal Engine 5を使用して開発されているため、PCプレイヤーは高いハードウェア要件に備える必要があります。Fictionsやゲームフリークによる公式スペックはまだ確定していませんが、初期の映像で見られるグラフィックの質や、標準的なUnreal Engine 5のパフォーマンスプロファイルに基づいて予測を立てることができます。

詳細な環境をシームレスに読み込むためには、SSD（ソリッドステートドライブ）が必須となる可能性が高いです。1440pや4K解像度で高いフレームレートを目指すプレイヤーは、高解像度のテクスチャや複雑なライティングシステムを処理するために、十分なビデオメモリを搭載した最新のグラフィックスカードが必要になるでしょう。

| ハードウェア構成 | 最小推定スペック | 推奨推定スペック |
| :--- | :--- | :--- |
| **プロセッサ (CPU)** | Intel i5 10400F / AMD Ryzen 5 3600 | Intel i7 10700K / AMD Ryzen 7 5800X |
| **システムメモリ** | 8 GB RAM | 16 GB RAM |
| **グラフィックスカード (GPU)** | GTX 1660 Super / RX 5600 XT | RTX 3070 / RX 6800 XT |
| **ビデオメモリ** | 6 GB VRAM | 10 GB GDDR6 VRAM |
| **ストレージ容量** | 50 GB HDD空き容量 | 75 GB SSD空き容量 |

<Callout type="success" title="プラットフォームの互換性">
このゲームは、Xboxエコシステムを通じてXbox Series X/SとPC間でのクロスプログレッシブ機能をサポートする予定です。これにより、Game Pass Ultimateユーザーはプラットフォームをシームレスに切り替えてプレイできます。
</Callout>
