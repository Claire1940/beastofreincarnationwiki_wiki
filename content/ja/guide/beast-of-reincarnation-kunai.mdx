<Section icon={Target} title="苦無マスターへのステップバイステップガイド">

**ビースト・オブ・リンカネーションの苦無**の可能性を完全に引き出すには、特定の進行経路を辿る必要があります。これには「日本の自然美」をテーマにしたバイオームから材料を集め、地殻の下に隠されたSF研究所でそれらを精製する工程が含まれます。

<Steps>
  <Step num="1" title="プロトタイプ・ブレードの入手">
    始まりの村でクエスト「過去の影（Shadows of the Past）」を完了します。これにより、エマとブラッドが使用する基本的な苦無セットが手に入ります。
  </Step>
  <Step num="2" title="輪廻の欠片を稼ぐ">
    廃墟の骨格（Skeleton Ruins）へ向かいましょう。機械の守護者を倒して「欠片（Shards）」を集めます。最初の主要なアップグレードには約50個の欠片が必要です。
  </Step>
  <Step num="3" title="属性スロットの解放">
    セントラルハブにあるメンテナンスステーションを訪れます。集めた欠片を使用して、苦無の「属性注入（Elemental Infusion）」スロットを解放します。
  </Step>
  <Step num="4" title="跳ね返り技術の習得">
    壁に対して45度の角度で苦無を投げる練習をしましょう。UE5の物理エンジンでは、飛び道具が勢いを失うまでに最大3回まで跳ね返ることが可能です。
  </Step>
</Steps>

| アップグレードレベル | 必要素材 | 解放される新能力 |
| :--- | :--- | :--- |
| **ランク 1** | 鉄の屑 x10 | 飛行速度の向上 |
| **ランク 2** | 蔦の繊維 x5 | 毒の塗布（継続ダメージ） |
| **ランク 3** | コアの破片 x2 | 二刀流が可能に |
| **ランク 4** | 輪廻の魂 x1 | 無限弾薬（クールダウン制） |

</Callout>

<Section icon={Sparkles} title="高度な戦術と属性付与">

基本をマスターしたら、世界中で見つかる様々な属性を**ビースト・オブ・リンカネーションの苦無**に注入できます。これらの注入は苦無の挙動を変化させ、ゲームの困難なボス戦を攻略するために不可欠です。

**属性シナジー表:**

| 属性 | 命中時の効果 | 有効な敵 |
| :--- | :--- | :--- |
| **電撃** | 近くの敵に連鎖する | 機械の守護者 / ロボット |
| **凍結** | 移動速度と攻撃速度を低下させる | 高機動のビーストボス |
| **ヴォイド** | 敵を小さな渦に引き寄せる | 小さな「蔦」クリーチャーの群れ |
| **火炎** | 累積する燃焼デバフを付与する | 巨大で動かない生物学的標的 |

これらの属性を環境と組み合わせて使用することが核心的なメカニクスです。例えば、地下廃墟の水たまりに電撃苦無を投げ込むと、広範囲にわたる衝撃波が発生します。この相互作用は、ギア・プロジェクトのエンジンにおける高度な物理実装の直接的な成果です。

<Callout type="info" title="エキスパート戦略">
エマとしてプレイする場合、ヴォイドクナイを使って敵をまとめ、その直後にシャドウダッシュを繰り出すことで、群れ全体に強力なクリティカルストライクを与えることができます。
</Callout>

</Section>

<Section icon={Shield} title="防御的ユーティリティと生存戦略">

苦無は主に攻撃的な武器ですが、初心者が見落としがちな防御的なレイヤーもいくつか提供しています。『ビースト・オブ・リンカネーション』では、特に高難易度の「リバース（再誕）」モードにおいて、生存はダメージ出力と同じくらい重要です。

- **パリィウィンドウ**: 飛んでくる飛び道具に対し、着弾直前に苦無を投げることで弾き飛ばすことができます。
- **注意逸らし**: 警戒していない敵の近くに苦無を投げると、敵はその音を調査しに行くため、隠密に通り抜けることができます。
- **脱出**: 「グラップルフック」属性を使用すると、壁に刺さった苦無に向かって自分を引き寄せ、危険な地面の効果から逃れることができます。

| 防御アクション | 入力要件 | 結果 |
| :--- | :--- | :--- |
| **弾き（Deflect）** | 投擲 + 精確なタイミング | 飛び道具を送り主に跳ね返す |
| **煙幕（Smoke Screen）** | スキル + 苦無2本 | 5秒間の不可視ゾーンを作成 |
| **クイックリカバリー** | 回避 + 苦無投擲 | ひるみアニメーションを即座にキャンセル |

<Callout type="warning" title="重要">
重いボスの攻撃を弾くと、通常の2倍の耐久度を消費します。ボスエリアに入る前には必ずメンテナンス状態を確認してください。
</Callout>

</Section>
