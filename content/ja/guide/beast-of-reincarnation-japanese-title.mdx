- エマとクゥによる**連携攻撃（チームコンバット）**のデモンストレーション。
- **サクラジャベリン**などのボス由来の特殊能力の初公開。

<Section icon={Swords} title="戦闘の極意：ブルームアーツとパリィ">

**ビースト・オブ・リンカネーション**のゲームプレイの核心は、そのハイブリッドな戦闘システムにあります。移動や基本攻撃はスピーディーなリアルタイム進行ですが、ブルームアーツを通じて「タクティカルポーズ（戦術的中断）」メカニクスが導入されています。ブルームアーツを発動すると時間の流れが極端に遅くなり、敵の状態に合わせて最も効果的な技を選択することができます。

戦闘を有利に進めるには、主に3つのゲージを管理する必要があります：

| ゲージの種類 | 役割 | 回復方法 |
| :--- | :--- | :--- |
| **体力ゲージ** | 生命力の総量を表す。 | 回復アイテムや特定のブルームアーツを使用。 |
| **ダウンゲージ** | 最大まで溜まると「ファイナルテイクダウン」が可能。 | 継続的なダメージや重攻撃を与える。 |
| **FPゲージ** | クゥの力やブルームアーツのエネルギー源。 | 敵の攻撃を完璧にパリィする。 |

<Steps>
  <Step num="1" title="パリィを完璧にする">
    敵の予備動作を待ちます。タイミングよくパリィを成功させると、すべてのダメージを無効化し、FPゲージが急速に溜まります。
  </Step>
  <Step num="2" title="ブルームアーツの発動">
    FPが満タンになったら、ブルームアーツメニューを起動します。時間がスローになり、敵の弱点やステータス異常を確認する余裕が生まれます。
  </Step>
  <Step num="3" title="スタンゲージを活用する">
    「バインディングルーツ」などの特定のスキルを使用して敵を拘束します。これによりスタンゲージが溜まり、一撃必殺のチャンスが生まれます。
  </Step>
  <Step num="4" title="連携フィニッシュの実行">
    エマの攻撃とクゥの追撃（「ヘルスピリッター」など）を連携させ、ボス級の敵に最大ダメージを与えます。
  </Step>
</Steps>

<Callout type="warning" title="パリィのタイミング">
ボスの強攻撃も受け流すことが可能ですが、タイミングの判定は通常の敵よりもはるかに厳しくなっています。ボスのパリィに失敗すると、大幅に体力を削られることが多いため注意が必要です。
</Callout>

</Section>

<Section icon={Target} title="キャラクターの相乗効果：エマとクゥ">

エマと愛犬クゥの関係は、単なるストーリー上の演出ではなく、ゲームのメカニクスの根幹です。プレイヤーは一人のキャラクターを育てるだけでなく、その「パートナーシップ」を進化させていきます。エマが精密な攻撃とブルームアーツの発動を担当し、クゥはサポート、クラウドコントロール（敵の制圧）、そして強力なフィニッシュ攻撃を繰り出します。

<CardGrid cols={3}>
  <Card title="エマ（追放者）">
    - **メインアタッカー**: 俊敏な武器を使い、素早いコンボを繰り出す。
    - **テクニックマスター**: 倒したボスから新しいブルームアーツを習得する。
    - **役割**: 高いDPSと戦術的な意思決定。
  </Card>
  <Card title="クゥ（相棒）">
    - **サポート＆ユーティリティ**: 最大3体のターゲットを同時に拘束可能。
    - **フィニッシャー**: 連携攻撃でとどめの一撃を見舞う。
    - **役割**: 敵の制圧と防御的な支援。
  </Card>
  <Card title="統合されたシナジー">
    - **共有SP**: 両キャラクターは共通のスキルツリーから成長する。
    - **ダイナミックコンバット**: クゥはエマの位置取りに合わせて自動的に反応する。
    - **役割**: 連携による戦術的な深みの最大化。
  </Card>
</CardGrid>

スキルツリーでは幅広い試行錯誤が可能です。冒険中に獲得したSPを消費してエマの新しい技を解放すると、それに続くクゥの追撃も強化されます。例えば「サクラジャベリン」をアップグレードすると、エマが遠距離から仕掛け、クゥが「ヘルスピリッター」で距離を詰めてとどめを刺すといった連携が可能になります。

| スキルタイプ | 主な効果 | 最適な用途 |
| :--- | :--- | :--- |
| **バインディングルーツ** | 複数の敵を拘束する。 | 大勢の敵を管理する際。 |
| **サクラジャベリン** | 高威力の遠距離貫通攻撃。 | 戦闘の開始時や飛行する敵に対して。 |
| **ヘルスピリッター** | 強力な振り下ろし物理攻撃。 | 敵のガードやアーマーを破壊する際。 |
| **バンバナルーツ** | 同時に複数のターゲットを制圧。 | ブルームアーツの隙を作る際。 |

<Callout type="tip" title="スキルツリーの戦略">
序盤はクゥの追撃スキルの解放を優先しましょう。これらの「無料」の追加ヒットは、FPを消費することなくダメージ出力を大幅に向上させます。
</Callout>

</Section>

<Section icon={Sparkles} title="世界観の美学と比較">

**ビースト・オブ・リンカネーション**のアートディレクションは、その「ジブリ風」の魅力で高い評価を得ています。4026年の日本の青々と茂る廃墟と、そこに潜む不気味で疫病に侵された獣たちとのコントラストが、独自の視覚的アイデンティティを生み出しています。批評家はしばしば他のAAAタイトルと比較しますが、本作は独自の戦闘メカニクスによって独自の地位を築いています。

| 比較対象タイトル | 本作との共通点 | 主な違い |
| :--- | :--- | :--- |
| **もののけ姫** | アートスタイル、環境テーマ、動物の相棒。 | ポストアポカリプス設定 vs 歴史ファンタジー。 |
| **SEKIRO: SHADOWS DIE TWICE** | パリィの重視、体幹/ダウンゲージの仕組み。 | 本作にはタクティカルポーズメニューがある。 |
| **黒神話：悟空** | 高精細なアクションとボス中心のゲームプレイ。 | 二人一組のシナジーと犬の相棒に焦点。 |
| **ELDEN RING** | オープンワールドの探索と発見。 | より物語主導で焦点を絞ったRPG構造。 |

世界観の構築において探索は非常に重要です。各エリアには門番としての強力なボスが存在します。これらの存在を倒すことは単に戦利品を得るだけでなく、エマの能力を物理的に変化させます。エマは倒したボスの力を吸収し、その敵の属性や性質を反映した新しいブルームアーツを覚醒させます。

<Callout type="success" title="探索の報酬">
ポストアポカリプスの廃墟を隅々まで探索すると、隠されたSPキャッシュや、4026年の疫病の起源を解き明かすロア（物語）の断片が見つかることがあります。
</Callout>

</Section>
