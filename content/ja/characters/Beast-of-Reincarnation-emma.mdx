<Section icon={Swords} title="戦闘とパリィシステムの極意">

**『Beast of Reincarnation』**の戦闘は、ハイスピードなアクションと戦術的な意思決定が融合したユニークなシステムです。エマをリアルタイムで操作する一方で、クゥのスキルを使用する際には意図的にアクションを一時停止させる必要があります。戦闘を有利に進める鍵は、エマの物理攻撃とクゥの超自然的な能力を繋ぐ「ジャストガード（パリィ）」システムにあります。

| 戦闘アクション | メカニクスタイプ | 効果 / メリット |
| :--- | :--- | :--- |
| **弱/強攻撃** | リアルタイム | 継続的なダメージと敵へのプレッシャー |
| **ジャストガード（パリィ）** | タイミング重視 | ダメージを無効化し、**コマンドポイント**を生成 |
| **クゥのコマンドメニュー** | 戦術的一時停止 | 時間の流れを遅くし、相棒の技を選択 |
| **植物髪の鞭** | リアルタイム | 中距離の範囲攻撃および間合いの瞬時の縮小 |
| **落下暗殺** | ステルス | 高所から仕掛ける高威力の先制攻撃 |

<Callout type="warning" title="パリィのタイミング">
パリィは単なる防御手段ではありません。主要なリソース生成アクションです。ジャストガードを成功させなければ、ボス戦でクゥの最も強力な技を発動することができなくなります。
</Callout>

戦闘の流れは基本的に特定の基本リズムに従います。敵の動きを観察し、攻撃をパリィし、コマンドメニューを開いてクゥの技を発動し、生まれた隙を突いて追撃します。

</Section>

<Section icon={Zap} title="エマの最強ビルドとおすすめ装備">

**『Beast of Reincarnation』**におけるキャラクターカスタマイズは、「刀」「魂石」「クゥの護符」という3つの核となる要素のバランス調整によって行われます。選択するビルドによって、瘴気への対処法や効果的な戦闘スタイルが変化します。

<Steps>
  <Step num="1" title="ベースとなる武器の選択">
    プレイスタイルに合った「刀」を選びます。デジタルデラックス版特典の**「北斗の剣（Big Dipper）」**はバランスの取れた性能を誇り、探索で手に入る刀は速度やパリィ受付時間の延長などに特化しています。
  </Step>
  <Step num="2" title="魂石の装着">
    魂石は特定のアクションによって発動するパッシブボーナスを付与します。攻撃的なビルドを目指すなら、ジャストガード成功時に攻撃力バフを付与する石を重視しましょう。
  </Step>
  <Step num="3" title="クゥの護符の装備">
    護符はクゥのコマンドスキルを変化させます。攻撃の攻撃範囲を広げるものや、防御スキルのコマンドポイント消費量を軽減するものなどがあります。
  </Step>
  <Step num="4" title="スキルツリーの解放">
    エマのパリィとクゥの追撃のシナジーを高めるため、スキルツリーの「共鳴（Resonance）」ブランチを優先的に強化しましょう。
  </Step>
</Steps>

| ビルドスタイル | 主なフォーカス | 主要装備 / スキル |
| :--- | :--- | :--- |
| **アグレッシブ近接** | 高DPS | 手数の多い刀、攻撃バフ付与の魂石 |
| **タクティカル防御** | パリィ重視 | 重厚な護符、パリィ受付時間延長スキル |
| **レンジサポート** | クゥのコマンド | コマンドポイント回復の魂石、範囲拡大の護符 |
| **ステルスストーカー** | 暗殺 | 垂直移動スキル、移動速度上昇の魂石 |

<Callout type="tip" title="ビルドの柔軟性">
スキルツリーは特定の「集落（コロニー）」でリセット可能です。新しい「主（ヌシ）」に挑む際は、恐れずに様々な魂石の組み合わせを試してみましょう。
</Callout>

</Section>

<Section icon={Map} title="探索とワールド移動">

4026年の日本は、垂直方向に広がる変化に富んだ構造をしています。エマの植物のような髪は、旧世界の廃墟を探索するための最大のツールです。崩落した橋を渡ることから、破壊された超高層ビルの壁を登ることまで、探索は戦闘と同じくらい重要な要素です。

| 移動アクション | 説明 | ゲームプレイでの用途 |
| :--- | :--- | :--- |
| **蔦グラップル** | 髪を伸ばして特定のポイントに引っ掛ける | 広い裂け目の横断や廃墟の登攀 |
| **壁走り** | 植物と融合した手足でグリップ力を高める | 狭い都市の路地裏の移動 |
| **瘴気変貌** | 環境がリアルタイムで変化する | 森の出現によって新たなルートが切り開かれる |
| **隠しルート** | 主要道路の上方に位置する垂直な経路 | レアな刀や魂石の発見 |

<Callout type="tip" title="上を見よ">
ゲーム内の古い伝承では「決して空を見てはならない」と警告されています。しかし、エマの垂直移動能力を駆使して地上より高い位置を維持することは、不要な禍獣との遭遇を避ける最も安全な方法です。
</Callout>

</Section>
