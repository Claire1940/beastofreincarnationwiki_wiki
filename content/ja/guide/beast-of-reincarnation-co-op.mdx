- 本作は、リアルタイムのハックアンドスラッシュと戦術的なターン制要素を融合させた**ハイブリッド戦闘システム**を採用しています。
- プレイヤーは**クゥのゲージメーター**を管理し、激しい戦闘中に強力な「ブルーミングアーツ（咲き誇る技）」を放ちます。
- クゥに**傍に控えさせる**ことも、**独立して突撃させる**ことも可能で、柔軟な戦闘戦略を立てることができます。
</Callout>

<Section icon={Map} title="相棒のメカニズム">

多くのプレイヤーが従来のマルチプレイを期待するかもしれませんが、**『ビースト・オブ・リンカネーション』**は革新的な「一人と一匹」の体験を提供します。一般的な協力プレイ（Co-op）とは異なり、本作は主人公のエマと、その相棒である狼のクゥとの絆に焦点を当てています。

戦闘システムは非常に戦術的です。戦闘中にクゥをいかに効果的に活用できるかが勝利の鍵となります。敵の攻撃をパリィすることでリソースゲージが溜まり、時間を遅らせて特殊なブルーミングアーツを選択し、戦況を覆すことができます。

| 特徴 | 説明 |
| --- | --- |
| 戦闘スタイル | リアルタイムアクションとターン制の戦術要素の融合 |
| 相棒の役割 | クゥは指示可能なサポート役およびアタッカーとして機能 |
| リソースシステム | パリィによりブルーミングアーツ用ゲージが蓄積 |
| コアメカニズム | 特殊技を発動するための時間減速タクティカルメニュー |

<Callout type="tip" title="戦略ノート">
強敵と対峙する際は、常にパリィを最優先にしてください。早い段階でクゥのゲージを溜めることで、どんなに手強い敵をも怯ませる高威力のブルーミングアーツを放つことができます。
</Callout>

装備システムでは、パッシブボーナスを得るために「精霊石」を使用します。これらの石は衣装（アウトフィット）に装着することができ、レベルアップさせることでその効果を高めることができます。

| 構成要素 | 機能 |
| --- | --- |
| 刀 | エマの主力近接武器 |
| 弓/クロスボウ | 遠距離戦闘用のツール |
| 精霊石 | ステータスやアビリティのパッシブ補正 |
| 衣装 | 精霊石のスロット容量を決定 |

<Callout type="warning" title="重要">
エマのような「封印者（シーラー）」は人々から恐れられていることを忘れないでください。都市部を探索する際は、自然を操る能力が不要な注目を集めてしまう可能性があるため、立ち回りに注意しましょう。
</Callout>

  </Step>
  <Step num="2" title="環境の探索">

  <Step num="3" title="ボス戦の準備">

  </Step>
</Steps>
