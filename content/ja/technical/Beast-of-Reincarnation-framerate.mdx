<Section icon={Sparkles} title="GPUコントロールパネルの最適化">

グラフィックスカードのコントロールパネルを最適化することは、**ビースト・オブ・リンカネーションのフレームレート**を向上させる最も直接的な方法です。GPUドライバーの設定で画質よりもパフォーマンスを優先することで、視覚的な体験を大きく損なうことなく、大幅なフレームレートの向上を実現できます。

**動画のハイライト:**
- **Nvidiaコントロールパネル**: グローバル3D設定を「クオリティ」から「パフォーマンス」に切り替える。
- **解像度のダウンスケーリング**: ネイティブ解像度をわずかに下げることで、大幅なFPS向上。
- **ディスプレイの整合性**: スケーリングによるラグを防ぐため、Windowsの解像度をゲーム内設定と一致させる。

<Section icon={Target} title="ランチャーと互換性のステップバイステップ設定">

ゲームランチャーのプロパティを構成することで、オペレーティングシステムがゲームに管理者権限を付与し、マイクロスタッターの原因となる不要な全画面表示の最適化レイヤーをバイパスできるようにします。

<Steps>
  <Step num="1" title="ランチャーのプロパティにアクセス">
    ゲームランチャー（Steamなど）を見つけ、アイコンを右クリックして**プロパティ**を選択します。
  </Step>
  <Step num="2" title="互換性設定の変更">
    **互換性**タブに移動します。**管理者としてこのプログラムを実行する**にチェックを入れます。
  </Step>
  <Step num="3" title="全画面表示の最適化を無効にする">
    **全画面表示の最適化を無効にする**にチェックを入れます。これにより、Windowsがゲーム独自のディスプレイ表示を管理できるようになり、入力遅延が減少します。
  </Step>
  <Step num="4" title="適用して再起動">
    **適用**をクリックし、次に**OK**をクリックします。起動前にすべての管理者権限が正しく登録されるよう、PCを再起動してください。
  </Step>
</Steps>

| 互換性設定 | 状態 | 目的 |
| :--- | :--- | :--- |
| **管理者として実行** | 有効 | システムリソースへのフルアクセスを許可 |
| **全画面表示の最適化を無効にする** | 有効 | マイクロスタッターと入力遅延を軽減 |
| **互換モード** | Windows 8（任意） | 旧エンジンによるクラッシュを解決 |

<Callout type="info" title="管理者モード">
ランチャーを管理者として実行することで、バックグラウンドソフトウェアがゲームによる重要なハードウェアスレッドへのアクセスをブロックするのを防ぐことができます。
</Callout>

</Section>

<Section icon={Shield} title="バックグラウンドアプリケーションとタスクマネージャーの調整">

実行中のバックグラウンドタスクが多すぎることが、突然のフレームレート低下の主な原因です。スタートアップリストを整理し、プロセスの優先度を調整して、スムーズなゲームプレイを保証する必要があります。

| プロセスの操作 | 推奨設定 | 節約されるリソース |
| :--- | :--- | :--- |
| **Xbox Game Bar** | 無効 | バックグラウンドCPU / RAM |
| **バックグラウンド録画** | 無効 | GPUエンコーダーの負荷 |
| **スタートアップアプリ** | 不要な項目をすべて無効化 | システム起動時のメモリ |
| **ゲームの優先度** | 高 または 通常以上 | CPUコアのスケジューリング |

<Callout type="success" title="優先度のヒント">
タスクマネージャーの詳細タブでゲームプロセスの優先度を「高」に設定すると、CPUがバックグラウンドのシステムタスクを処理する前にゲームの命令を処理するようになります。
</Callout>

</Section>
