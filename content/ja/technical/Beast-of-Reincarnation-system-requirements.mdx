パリィを成功させることで、コーのゲージが溜まります。発動すると時間の流れが遅くなり、ラジアルメニューから強力な「ブルーミングアーツ」を選択できます。このハイブリッドシステムは、素早いアクションと戦略的な意思決定を融合させており、現代のアクションRPGのフォーマットを彷彿とさせます。

| 戦闘メカニクス | 入力 / トリガー | ゲームプレイ効果 |
| :--- | :--- | :--- |
| **ジャストパリィ** | タイミングを合わせたL1/LBブロック | コーのゲージを溜め、敵の体幹を削る |
| **ブルーミングアーツ** | ラジアルメニューによる時間鈍化 | 甚大なダメージを与え、状態異常を付与する |
| **オーバードライブ** | L3 + R3 同時押し（ゲージ最大時） | 瞬間火力を高めるエンタングルメント・オーバードライブを発動 |
| **コンパニオンコマンド** | Lボタン + 十字キー | コーに待機やターゲットへの攻撃を指示する |
