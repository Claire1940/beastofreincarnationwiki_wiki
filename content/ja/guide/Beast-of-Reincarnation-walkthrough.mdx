<Section icon={Settings} title="キャラクターの育成とビルドのカスタマイズ">

『ビースト・オブ・リンカネーション』におけるキャラクターの育成はエマとクーに分かれており、それぞれが独立したスキルツリーとインベントリを持っています。戦闘を行うことでSP（スキルポイント）を獲得し、どちらのキャラクターにも自由に割り振ることができます。エマのスキルツリーは移動、体力、近接戦闘に焦点を当てており、クーのツリーは属性の威力やクラウドコントロールを強化します。

カスタマイズはスキルポイントだけに留まりません。エマは様々な剣を装備することができ、それぞれの剣には通常パリィ成功時に発動する固有のパッシブ効果が備わっています。さらに、「ソウルストーン」システムがビルド構築に奥深さを与えます。これらの石をエマの防具（ガンベゾン）に装着することで、クリティカル率の向上からパリィ受付時間の延長まで、幅広いバフを得ることができます。

<Section icon={Map} title="探索と移動のメカニクス">

『ビースト・オブ・リンカネーション』の世界は、単一のオープンワールドではなく、広大な「隔離ゾーン」の集合体で構成されています。これらのゾーン内の移動は非常に垂直的であり、エマの移動アビリティを創造的に活用することが求められます。レベルデザインはメトロイドヴァニアに近く、特定のアップグレードをアンロックするか、環境の物理挙動をマスターするまではアクセスできないエリアが多々存在します。

<Steps>
  <Step num="1" title="グラップルをマスターする">
    グラップルを使って高所の足場に到達したり、裂け目を飛び越えたりします。戦闘中に素早く位置取りを変えるためにも極めて有効な手段です。
  </Step>
  <Step num="2" title="植物の竹馬を展開する">
    植物の力を利用して、一時的な足場や橋を作り出します。危険な地形を渡ったり、隠された宝箱に到達したりするのに不可欠です。
  </Step>
  <Step num="3" title="高低差を活かす">
    本作には落下ダメージがありません。これを利用して、高い場所から敵に飛び降りてステルス処刑を仕掛けたり、危険な状況から離脱したりしましょう。
  </Step>
</Steps>

探索を進めることで、戦利品、クラフト素材、そして世界観を紐解くロア（伝承）が手に入ります。ただし、多くのゾーンは強力なエリアボスである「ヌシ」によって守られており、先へ進むには彼らを倒さなければなりません。これらのボスを倒すと新しいアビリティが手に入り、それによって前のゾーンを完全に探索するために必要だった移動スキルが手に入ることがよくあります。

| 移動ツール | 戦闘での実用性 | 探索での実用性 |
| :--- | :--- | :--- |
| **グラップル** | 迅速なポジショニング | 高いフロアへの移動 |
| **植物の竹馬** | 戦場のコントロール | 裂け目や穴の横断 |
| **植物の橋** | 高所の優位性確保 | ロックされたバルコニーへの侵入 |
| **ステルスカバー** | 視線の遮断 | ゴーレムの巡回を回避 |

<Callout type="info" title="落下ダメージなし">
高い崖から飛び降りることを恐れる必要はありません。落下ダメージがないため、垂直方向への自由な探索が推奨されており、敵の陣地へユニークなルートから奇襲をかけることができます。
</Callout>

</Section>

<Section icon={Zap} title="属性状態異常と敵のタイプ">

属性システムを理解することは、本作の攻略を成功させる上で極めて重要です。敵には体幹ゲージ（スタッガーメーター）があり、これを削りきることで壊滅的なダメージを与える処刑攻撃（エグゼキューション）が可能になります。このプロセスは、敵の弱点属性を突いて属性効果を付与することで加速させることができます。主な属性は「炎上（Burn）」「電撃（Shock）」「毒（Poison）」「腐食（Corrosion）」の4つです。

敵は大きく分けて「禍獣（マレファクト：汚染された生物）」と「ゴーレム（機械仕掛けの番兵）」の2つのカテゴリに分類されます。禍獣は炎上や毒に弱いことが多いですが、ゴーレムの重装甲を無力化するには電撃や腐食が必要になる場合があります。エリアボスである「ヌシ」はこのシステムの究極の試練であり、隙を作るために複数の状態異常を重ねて付与することが求められます。

| 属性 | 状態異常効果 | 特効対象 |
| :--- | :--- | :--- |
| **炎上** | 持続ダメージ | 高体力の禍獣 |
| **電撃** | 気絶と行動中断 | 動きの素早いゴーレム |
| **毒** | 徐々に体力を減少 | 有機物の敵の群れ |
| **腐食** | 装甲（防御力）低下 | 重装甲のヌシ（ボス） |

<Callout type="tip" title="シナジーボーナス">
クーは多くの属性アビリティを独占的に使用できます。クーを使って電撃などの状態異常を付与し、すかさずエマの重攻撃で追撃することで、体幹ゲージの蓄積を最大化しましょう。
</Callout>

</Section>

<Section icon={Trophy} title="移動拠点（メック）：クラフトと研究">

メック（巨大ロボット）は、隔離ゾーン間を移動するモバイル作戦基地として機能します。ここでは、戦闘以外の育成プロセスの大部分が行われます。メックの内部では、一時的なバフを得るための料理、新しい装備のクラフト、そしてメック自体の機能を拡張するためのアップグレード研究を行うことができます。

メック基地での機能拡張には、探索中に集めた素材が必要です。料理の食材はフィールドで採集でき、クラフト素材は倒したゴーレムからドロップするか、ロックされた宝箱から入手できます。隔離ゾーンの難易度が上がるにつれて、装備を適正レベルに保つために、新しいレシピや設計図の研究が不可欠になります。

| 基地施設 | 活動内容 | 主な成果物 |
| :--- | :--- | :--- |
| **キッチン** | 料理 | 一時的なステータスバフや回復アイテム |
| **鍛冶場** | クラフト/強化 | 高火力の武器やより優れた防具 |
| **研究ラボ** | 設計図の解析 | 新しいレシピやメックの機能拡張 |
| **休憩エリア** | 休息 | 世界の敵のリセットと体力の全回復 |

<Callout type="info" title="メックのアップグレード">
メック自体のアップグレードも怠らないようにしましょう。メックを強化することで、新しいファストトラベルポイントがアンロックされたり、エマとクーがフィールドに出ている間も適用されるパッシブボーナスを得られたりします。
</Callout>

</Section>
